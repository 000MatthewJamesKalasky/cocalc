Salvus™ distributed scalable online mathematical software

About

Description

  • Salvus provides mathematical software as a service: the main user language is Python as provided by Sage, which in turns provides access to many other programs.
  • Salvus has a distributed and fault tolerant design: it runs on hardware located in two separate data centers at the University of Washington, and on cloud hosting providers including Servedby.net.
  • Many features are planned for the first beta release of Salvus in January 2013: version control, simultaneous coordinated use by all students in a course, embedding in external websites, very long-running and large-memory research computations, and mobile clients.

Timetable

  • Salvus is currently freely available as alpha quality software, which is undergoing rapid development and may be very unstable.
  • The first beta release will be in January 2013, and the first stable release is slated for March 2013.
